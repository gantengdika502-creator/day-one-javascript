# Tugas Web Dasar - Implementasi JavaScript

Repository ini berisi tugas sekolah tentang berbagai metode penempatan dan penggunaan JavaScript dalam halaman web.

Project ini terdiri dari 5 halaman utama yang masing-masing menggunakan metode berbeda:

* **External (`index.html`)**: Memisahkan kode logika ke file `script.js`.
* **Internal (`internal.html`)**: Menaruh kode JavaScript di dalam tag `<script>` pada satu file.
* **Embed (`embed.html`)**: Mirip dengan internal, digunakan untuk kode yang menyatu dengan halaman.
* **Inline (`inline.html`)**: Menaruh perintah JavaScript langsung di atribut tag HTML (seperti `onclick`).
* **Prompt (`prompt.html`)**: Menggunakan fungsi `prompt()` untuk mengambil input dari pop-up browser.
